fn execute(
    &mut self,
    sender: Addr,
    msg: CosmosMsg<C>
) -> Result<AppResponse, Error>

Runs arbitrary CosmosMsg. This will create a cache before the execution, so no state changes are persisted if this returns an error, but all are persisted on success.
