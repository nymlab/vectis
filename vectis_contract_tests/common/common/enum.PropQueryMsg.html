ListProposals

§start_after: Option<u64>

The proposal ID to start listing proposals after. For example, if this is set to 2 proposals with IDs 3 and higher will be returned.

§limit: Option<u64>

The maximum number of proposals to return as part of this query. If no limit is set a max of 30 proposals will be returned.

Lists all the proposals that have been cast in this module.

The proposal ID to start listing proposals before. For example, if this is set to 6 proposals with IDs 5 and lower will be returned.

§limit: Option<u64>

The maximum number of proposals to return as part of this query. If no limit is set a max of 30 proposals will be returned.

Lists all of the proposals that have been cast in this module in decending order of proposal ID.
