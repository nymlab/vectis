§exec_plugins: Map<'a, &'a str, PluginInfo>

Plugins that can exec through this contract unconditionally. The plugin should be from a trusted source managing all permissions This is similar to granting authz to the plugin Example of this: automation, allowance, session keys (assuming each plugin can only have 1 permission) We separate these so that we can only loop through the required ones at pre / post hooks
